# SkillHub Manager

> 把 Claude Code Skills 变成可运营资产。

SkillHub Manager 是一款面向 Claude Code 用户的本地 Skills 管理桌面工具，提供可视化管理、智能推荐、快速启停与安全审计能力。

## 下载

- [macOS 版（DMG）](https://github.com/GitHubDiom/SkillHub/releases/download/v1.5.0/SkillHub-Manager-mac.dmg) — v1.5.0 测试版
- Windows / Linux 版本即将推出

## 核心功能

- **Skills Library** — 自动扫描本地 Skills，表格化展示，支持搜索、分类、收藏、启用/禁用与回收恢复
- **Market** — 内置浏览器直接预览 GitHub 仓库，智能识别仓库类型（单 Skill / 集合 / 插件 / 混合），一键 Git Clone 安装
- **Statistics** — 总量、分类、标签与评分一目了然，数据驱动管理决策
- **Integration** — 一键生成 Claude Code / Codex / Cursor 集成配置
- **GitHub Device Flow** — 安全登录，无需暴露密钥，提升 API 配额

## 技术栈

Electron + React + TypeScript + SQLite + Vite
